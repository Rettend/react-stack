## Getting Started

> [!TIP]
> Use [ni](https://github.com/antfu-collective/ni):
>
> ```bash
> npm i -g @antfu/ni
> ```

### Install dependencies

```bash
ni
```

## Scripts

### Run development server

```bash
nr dev
```

### Build for production

```bash
nr build
```

### Preview production build

```bash
nr preview
```

### Lint and format

```bash
nr lint
```
