# React Stack

React, TypeScript, Vite, Bun, UnoCSS, ESLint (antfu), Shadcn UI, Iconify, Fontsource

## Getting Started

> [!TIP]
> Use @antfu/ni
>
> ```bash
> npm i -g @antfu/ni
> ```

### Install dependencies

```bash
ni
```

## Scripts

### Run development server

```bash
nr dev
```

### Build for production

```bash
nr build
```

### Preview production build

```bash
nr preview
```

### Lint and format

```bash
nr lint
```
